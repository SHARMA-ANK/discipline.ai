# DisciplineAI

A strict AI-powered productivity coach built with Flutter, Supabase, and Google Gemini.

##  Features
- **Strict AI Coach:** Roasts you if you miss tasks.
- **Clean Architecture:** Scalable and testable code.
- **Supabase Backend:** Auth, Database, and Edge Functions.
- **Google Gemini:** The brain behind the coach.

##  Setup
1. Clone the repo.
2. Add your Supabase keys in lib/core/constants/supabase_constants.dart.
3. Run flutter run.
